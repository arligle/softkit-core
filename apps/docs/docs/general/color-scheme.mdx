### Light Theme Colors

- **Primary Color (`--primary`):** <span style={{ backgroundColor: '#f86b00', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#f86b00</span> - Vibrant orange, used for primary actions and highlights.
- **Secondary Color (`--secondary`):** <span style={{ backgroundColor: '#df6000', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#df6000</span> - Darker shade of orange for secondary elements.
- **Tertiary Color (`--secondary-light`):** <span style={{ backgroundColor: '#ff9443', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#ff9443</span> - Lighter orange for background accents.
- **Dark Orange (`--dark-orange`):** <span style={{ backgroundColor: '#d35b00', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#d35b00</span> - For darker accents.

### Dark Theme Colors

- **Background Color (`--ifm-background-color`):** <span style={{ backgroundColor: '#1A1818', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#1A1818</span> - Main background color.
- **Navbar Background (`--ifm-navbar-background-color`):** <span style={{ backgroundColor: '#1f1d1e', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#1f1d1e</span> - Navbar background.
- **Text Color (`--ifm-font-color-base`):** <span style={{ backgroundColor: '#ffffff', color: '#000', padding: '3px 8px', borderRadius: '4px' }}>#ffffff</span> - Primary text color.
- **Heading Color (`--ifm-heading-color`):** <span style={{ backgroundColor: '#ffffff', color: '#000', padding: '3px 8px', borderRadius: '4px' }}>#ffffff</span> - Color for headings.

### Neutral and Text Colors

- **Primary Text Color (`--black100`):** <span style={{ backgroundColor: '#1f1d1e', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#1f1d1e</span> - Used for primary text.
- **Secondary Text Color (`--black60`):** <span style={{ backgroundColor: '#d35b00', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#d35b00</span> - Used for secondary text.
- **Backgrounds and Borders (`--black10`):** <span style={{ backgroundColor: '#FEE1CC', color: '#000', padding: '3px 8px', borderRadius: '4px' }}>#FEE1CC</span> - Used for backgrounds and borders.

### Accent Colors

- **Secondary Light Blue (`--secondary-light-blue`):** <span style={{ backgroundColor: '#20a4f3', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#20a4f3</span> - Bright blue for links and interactive elements.
- **Positive Default (`--positive-default`):** <span style={{ backgroundColor: '#31c587', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#31c587</span> - Green for success messages.
- **Warning Default (`--warning-default`):** <span style={{ backgroundColor: '#f6aa50', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#f6aa50</span> - Amber for warnings.
- **Negative (`--negative`):** <span style={{ backgroundColor: '#e93c51', color: '#fff', padding: '3px 8px', borderRadius: '4px' }}>#e93c51</span> - Red for errors.

These colors are integral to our design system, ensuring a cohesive and accessible user experience in both light and dark modes.
