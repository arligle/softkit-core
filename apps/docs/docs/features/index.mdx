# Available Features

Welcome to the Available Features section! Here you'll find information on the various tools and utilities we offer to streamline and enhance your development process. Explore our features to learn how they can help you build robust and efficient applications.

## Backend Client Generator

The Backend Client Generator is designed to help you create a client library that other backend services can use to make synchronous HTTP calls to your microservice. This generator simplifies the integration process and ensures seamless communication between services.

For more detailed information, visit the [Backend Client Generator](/features/code-generators/backend-client-generator) page.
